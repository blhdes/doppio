# App Store screenshots — Doppio

Apple needs screenshots before you can submit. Doppio is **iPhone-only**
(`TARGETED_DEVICE_FAMILY = 1`), so you only need **iPhone** shots — no iPad set.

## Required size

App Store Connect requires **one set of 6.9-inch iPhone screenshots**, portrait:

- **1320 × 2868** (iPhone 16 Pro Max), or
- **1290 × 2796** (iPhone 15/16 Pro Max) — either is accepted in the 6.9" slot.

- **Count**: 1–10. Aim for **5**.
- Optionally also upload a 6.5" set (1242 × 2688 or 1284 × 2778); not required.

> Don't have a 6.9" iPhone? Capture on whatever iPhone you have, then resize to
> exactly **1290 × 2796**. Every modern iPhone is ~0.461:1, the same aspect as
> 1290 × 2796, so upscaling Doppio's single full-screen layout looks clean. Tell me
> your shots' size and I'll resize them to the exact pixels for you.

## How to capture (on your device — no Simulator)

1. Open Doppio on your iPhone.
2. Set up the shot (theme / mode / beat style — see list below).
3. Press **Side button + Volume Up** together → the screenshot lands in Photos.
4. AirDrop or sync the images to your Mac.

The app already hides the status bar, so shots come out clean with no clock/battery.

## Suggested 5-shot list

Show range — different themes and beat styles, a steady readable number each time.

1. **Hero** — a bold dark theme, **Pulse** style, **DOUBLE** mode, a clean number (e.g. result 256 from 128). This is your first/most important shot.
2. **The flip** — **HALF** mode on a second theme, so the ×½ relationship reads.
3. **A different motion** — **Orbit** or **Sweep** mid-animation on a third theme.
4. **Fine-tune** — second finger down: the source number swelled on its glass capsule, with the right-edge BPM scale lit. Shows the precision feature.
5. **Range** — a **light** theme (to contrast the dark ones) and a fourth beat style, e.g. **Ripple** or **Bare**.

(Optional: add a short text caption above each in App Store Connect's media editor — but raw screenshots are perfectly acceptable for v1.)
